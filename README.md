# Flightscry ✈️

Flightscry is a modern, mobile-first flight itinerary proof of concept (PoC) built for Skyscanner. It provides a sleek, interactive interface for travelers to view their upcoming flight details, journey progress, and boarding passes.

## Features

- **Dynamic Flight Overview**: Real-time status, flight numbers, and airline information.
- **Interactive Itinerary**: Visual timeline of departure and arrival details, including terminal and gate information.
- **Glassmorphism UI**: A "crafted" interface using modern design principles like glassmorphism and soft shadows.
- **Animated Transitions**: Smooth entrance animations for cards and interactive elements using `motion`.
- **Boarding Pass Simulation**: A dedicated view for passenger boarding passes with simulated barcodes and verification status.
- **Amenity Tracking**: Quick glance at available in-flight services like Wifi, meals, and entertainment.

## Tech Stack

- **React 19**: Frontend framework.
- **TypeScript**: Type-safe development.
- **Tailwind CSS**: Utility-first styling with custom theme configurations.
- **Motion**: High-performance animations and transitions.
- **Lucide React**: Crisp, consistent iconography.
- **Vite**: Ultra-fast build tool and development server.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository (if applicable).
2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Start the development server:
```bash
npm run dev
```
The application will be available at `http://localhost:3000`.

### Building for Production

Create a production-ready build:
```bash
npm run build
```
The output will be in the `dist/` directory.

## Design Philosophy

Flightscry follows the **Skyscanner Backpack** design spirit, focusing on:
- **Clarity**: Information is scannable and prioritized.
- **Trust**: Clean typography and verified badges.
- **Delight**: Subtle animations and responsive interactions that make the app feel "alive".
